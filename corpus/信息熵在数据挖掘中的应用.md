# 信息熵在数据挖掘中的应用

## 基本概念

信息熵用于刻画概率分布的不确定性。一个分布越均匀，熵越高；一个分布越集中，熵越低。

在机器学习和数据挖掘中，信息熵常用于特征选择、决策树划分、异常检测和模型不确定性分析。

## 用于问答风险识别

在知识库问答中，可以把信息熵用于两个关键环节：

- 检索熵 H_ret：衡量检索结果是否集中在少数高相关证据上。如果多个文档分数接近，说明证据来源分散，答案可能不稳定。
- 输出熵 H_ans：衡量模型生成 token 时的概率分布是否分散。如果关键 token 的候选概率接近，说明模型在表达或事实选择上更不确定。

## 预期效果

信息熵研究希望达到的效果，是把模型回答前后的不确定性量化出来，并与冲突检测、敏感性检测结合，形成可解释的风险控制策略。

当熵较低且证据一致时，可以正常输出；当熵较高、证据缺失或来源冲突时，应提示不确定性、要求二次确认，严重时进入人工复核或拦截流程。
